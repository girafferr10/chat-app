6. OMEN SYSTEM (DoT ENGINE)
State
enemy.omen: int
Trigger (Enemy Turn Start)
def resolve_omen(enemy):

    dmg = enemy.omen * 1.2

    if enemy.type == "NORMAL":
        dmg *= 1.20
    elif enemy.type == "ELITE":
        dmg *= 1.10
    elif enemy.type == "BOSS":
        dmg *= 1.05

    enemy.hp -= dmg

    enemy.omen = ceil(enemy.omen / 2)
Rules
Only triggers on HIT
Has soft cap: 9999
Does NOT interact with crit directly
